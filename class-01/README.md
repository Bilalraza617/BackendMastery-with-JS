# How to Install Node.js and Run JavaScript Files 🚀

A complete step-by-step guide for beginners to install Node.js on Windows, macOS, and Linux and run their first JavaScript file.

---

## 📋 What is Node.js?

Node.js is a free, open-source, server-side JavaScript runtime environment. It allows you to run JavaScript outside the browser.

### Key Features:

- **Fast & Lightweight**
- **Built on Chrome’s V8 Engine**
- **NPM Included** (World’s largest package manager)
- Run JavaScript on the server

## Why Use Node.js?

- Build fast web servers and APIs
- Access thousands of open-source packages via npm
- Run JavaScript locally for scripting and automation
- Perfect for full-stack JavaScript development

---

## Step-by-Step Installation Guide

### Step 1: Download Node.js

1. Go to the **Official Website**: [https://nodejs.org](https://nodejs.org)
2. Download the **LTS Version** (Recommended for most users)
3. Save the installer

### Step 2: Install Node.js on Windows

1. Run the downloaded `.msi` installer
2. Accept the license and click **Next**
3. Keep default settings (recommended)
4. Click **Install**
5. Finish the setup

### Step 3: Verify Installation

Open Command Prompt or PowerShell and run:

```bash
node --version
npm --version
```
